## Category 2 — Firebase Readiness: 20%

### Passing ✅
- Firebase client code complete: `mobile/src/firebase.js` (auth, Firestore, all hooks)
- Backend Firebase middleware implemented: `backend/middleware/firebaseAuth.js`
- Firestore security rules documented in `mobile/BUILD.md`

### Missing / Broken ❌
- **Firebase project not created** — no `REACT_APP_FIREBASE_*` values set
- **`google-services.json` missing** — required in `mobile/android/app/` for Android Firebase SDK
- **`FIREBASE_SERVICE_ACCOUNT` not set** in backend `.env` — backend can't verify mobile tokens
- **`FIREBASE_PROJECT_ID` not set** in backend `.env`

### Manual steps required

1. **Create Firebase project:**
   - Go to [https://console.firebase.google.com](https://console.firebase.google.com)
   - Click **Add project** → Name: `jarvis-ai` (or `ooplix`)
   - Disable Google Analytics (optional)

2. **Enable Authentication:**
   - Firebase Console → **Authentication** → **Get Started**
   - Sign-in method → **Email/Password** → Enable

3. **Create Firestore database:**
   - Firebase Console → **Firestore Database** → **Create database**
   - Mode: **Production mode**
   - Region: `asia-south1` (Mumbai) — closest to your users

4. **Add Firestore security rules** (copy from `mobile/BUILD.md` Step 1)

5. **Add Web app (for mobile React):**
   - Project Settings → **Your apps** → **Add app** → **Web** `</>`
   - App nickname: `Ooplix Mobile`
   - Copy the `firebaseConfig` object

6. **Add Android app:**
   - Project Settings → **Your apps** → **Add app** → **Android** 🤖
   - Android package name: `com.jarvisai.app`
   - Download `google-services.json`
   - Place at: `mobile/android/app/google-services.json`

7. **Generate service account (for backend):**
   - Project Settings → **Service Accounts** → **Generate new private key**
   - Download JSON → convert to single line: `cat key.json | tr -d '\n'`

8. **Add to `mobile/.env`:**
   ```env
   REACT_APP_API_URL=https://app.ooplix.com
   REACT_APP_FIREBASE_API_KEY=AIzaSy...
   REACT_APP_FIREBASE_AUTH_DOMAIN=your-project.firebaseapp.com
   REACT_APP_FIREBASE_PROJECT_ID=your-project-id
   REACT_APP_FIREBASE_STORAGE_BUCKET=your-project.appspot.com
   REACT_APP_FIREBASE_MESSAGING_SENDER_ID=123456789
   REACT_APP_FIREBASE_APP_ID=1:123456789:web:abc123
   REACT_APP_VERSION=1.0.0
   ```

9. **Add to backend `.env`:**
   ```env
   FIREBASE_PROJECT_ID=your-project-id
   FIREBASE_SERVICE_ACCOUNT={"type":"service_account","project_id":"..."}
   ```

### Fix commands

```bash
# Install Android platform
cd mobile
npm install
npx cap add android

# Copy reference configs
cp android-config/AndroidManifest.xml android/app/src/main/AndroidManifest.xml
cp android-config/build.gradle.app android/app/build.gradle

# Place google-services.json (downloaded from Firebase)
cp ~/Downloads/google-services.json android/app/google-services.json

# Generate keystore (run ONCE — keep safe forever)
keytool -genkey -v \
  -keystore jarvis-release.keystore \
  -alias jarvis \
  -keyalg RSA -keysize 2048 -validity 10000
# Answer prompts, remember passwords

# Build and sync
npm run cap:sync
npm run cap:open   # opens Android Studio
```
