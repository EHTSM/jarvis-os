# Ooplix Academy

Learn Ooplix through structured learning paths. Complete lessons to earn badges and certificates.

## Learning Paths

### 1. Founder Fast-Track (2 hours)
*For solo founders who want to be productive immediately*

1. Quick Start — install, configure, first login
2. Your first Mission — describe a goal, watch it execute
3. CRM setup — import leads, configure WhatsApp follow-up
4. Billing — create your first payment link
5. Dashboard — understand your metrics

**Badge:** Founder Badge

---

### 2. Developer Workspace (3 hours)
*For developers who want to use Ooplix as their IDE*

1. Code editor — navigation, syntax highlighting, themes
2. AI pair programming — explain, refactor, generate, review
3. Visual Git — stage, commit, branch, PR workflow
4. Mission tracking — link commits to missions
5. Terminal — PTY, environment, scripts

**Badge:** Developer Badge

---

### 3. Growth Operating System (2 hours)
*For marketers and founders running campaigns*

1. Audience manager — import contacts, build segments
2. Email campaigns — create, test, send
3. WhatsApp broadcasts — templates, sequences
4. Marketing automation — visual flow builder
5. Analytics — open rates, clicks, revenue attribution

**Badge:** Growth Badge

---

### 4. AI Platform Architecture (4 hours)
*For technical founders building on top of Ooplix*

1. Smart router — how AI provider selection works
2. Agent runtime — planner/reviewer/verifier agents
3. Knowledge graph — nodes, relations, impact analysis
4. Plugin SDK — build custom integrations
5. API reference — authenticate, dispatch, stream

**Badge:** Platform Badge · **Certificate:** Ooplix Certified Developer

---

## How to access

1. Open Ooplix desktop app
2. Go to **Launch → Academy** tab
3. Select a learning path
4. Complete lessons at your own pace

Progress is saved automatically. Certificates are generated as PDFs.
